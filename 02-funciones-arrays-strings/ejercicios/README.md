# Ejercicios — Funciones, arrays y strings 🏋️

1. **Estadísticas de array** — dado `[4, 8, 15, 16, 23, 42]`, calcula suma, media, máximo y mínimo
   usando funciones de array (`array_sum`, `max`, `min`, `count`). Sin bucles manuales.
2. **Contador de palabras** — dada una frase, devuelve un array asociativo `palabra => veces`
   ordenado por frecuencia. Practica `preg_split`, `??` y `arsort`.
3. **Slug** — convierte `"Hola Mundo PHP!"` en `"hola-mundo-php"`. Es lo que hace `cocur/slugify`
   en Perfex para generar URLs limpias.

Soluciones en [`soluciones/`](soluciones/). Ejecuta cada una con `php soluciones/NN-xxx.php`.
