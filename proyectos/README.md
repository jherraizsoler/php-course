# 🏗️ Proyectos prácticos

Aquí juntas todo lo aprendido. Dos versiones del **mismo** proyecto (CRUD de tareas/usuarios) para
que veas la diferencia entre hacerlo "a mano" y hacerlo con un framework.

| Proyecto | Qué practica | Requiere |
|---|---|---|
| [`crud-php-puro/`](crud-php-puro/) | PDO, MVC manual, seguridad, sesiones | MAMP (Apache + MySQL) |
| [`crud-codeigniter3/`](crud-codeigniter3/) | CI3, Query Builder, validación, flashdata | MAMP + CodeIgniter 3 |

**Orden recomendado:** primero el de PHP puro (entiendes lo que pasa por debajo), luego el de CI3
(ves cuánto te ahorra el framework — y reconoces el patrón de enfoca-nexo).

Cada carpeta tiene su propio `README.md` con instrucciones paso a paso.
